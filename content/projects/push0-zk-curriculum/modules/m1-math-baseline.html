M1: Math Baseline for ZK

DFS prerequisites: finite fields, modular arithmetic, polynomials, FFT intuition, asymptotics, probability bounds.

Topics to teach (not assumed)

TopicWhy neededStop at
Finite fields F_pAll major proving systems operate over fields.Assume modular arithmetic arithmetic skill if comfortable; else refresh.
Polynomials and rootsConstraint systems are polynomial identities.Assume basic polynomial manipulation; teach interpolation/use in protocols.
Asymptotic complexityCompare prover/verifier/runtime tradeoffs.Assume basic big-O syntax; teach interpretation in proofs.
Probability for soundness errorZK security statements are probabilistic.Assume basic probability; teach negligible error meaning.

Learning outcomes

  • Explain why fields (not integers) are used in proving arithmetic.
  • Interpret statements like soundness error 2^-lambda.
  • Read runtime statements without hand-waving.

Exercises

  • Compute simple identities in F_17.
  • Given a polynomial constraint, test assignment validity.
  • Convert a protocol runtime statement into practical intuition.
