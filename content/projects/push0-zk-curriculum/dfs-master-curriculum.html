This is the deep version of the curriculum. Every module expands prerequisites recursively (DFS) until we hit a high-school baseline.

How this DFS curriculum works

  • Start from target topic: Understand and evaluate push0 real-time proving claims.
  • For each required concept, recursively add prerequisites.
  • Stop recursion when a prerequisite is usually high-school level (basic algebra, basic probability, basic functions/graphs).
  • Track each node as: Teach, Optional refresh, or Assumed high-school.
