# DeFi From First Principles: Beginner-Friendly Curriculum

If you are technical but not from finance, start here.

## How This Version Is Different

- Plain-language first, jargon second.
- Every core term is defined with a real-life analogy.
- One consistent story (Maya and Arjun) is used across modules.
- Each module now has:
  1. `visual.html` (diagram + summary)
  2. `deep-dive.html` (long-form explanation)
